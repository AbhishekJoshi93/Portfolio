Happy Home

Frontend & Backend Website

Tools

Html, Css, Js, Jquery, Ejs, Nodejs, Express, Passportjs, Mongodb, Mongoose, Stripejs, Socketio, Twilio, Multer, Heroku, Npm

Description

Happy Home is a home rental management system website where every user is authorized with passportjs. Every user can post there listing for sold or lease there home upto few number for unlimited listing they have buy vip account. Also vip user have more priority then common users listing in standing. Users can chat with each users and final the deal.
